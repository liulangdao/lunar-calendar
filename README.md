# 📅 农历黄历日历订阅源生成器 (Lunar & Huangli ICS Calendar)

这是一个基于 GitHub Actions 自动运行的中国农历、传统老黄历、节日与节气订阅源生成器。它每天定时（北京时间零点）计算并生成包含**前一年、今年、后一年（滚动共 3 年）**的 iCalendar (`.ics`) 文件。

你可以直接将其订阅链接导入到 iPhone、Mac、Google Calendar 或 Outlook 日历中，无需安装任何额外 App，直接在系统日历中查看每日黄历与传统节日信息。

## 🛠 订阅配置指南

### 📱 苹果设备 (iPhone / iPad / Mac Calendar)
1. 打开**日历** App。
2. **iPhone / iPad**: 点击底部“日历” -> “添加日历” -> “添加订阅日历”。
3. **Mac**: 点击顶部菜单栏 “文件” -> “新建日历订阅...”。
4. 粘贴您的订阅链接并点击确定。
5. 配置刷新频率（建议设置为“每天”），即可自动同步。

### 🌐 谷歌日历 (Google Calendar)
1. 打开 [网页版谷歌日历](https://calendar.google.com/)。
2. 在左侧“其他日历”旁边，点击 **`+`** 按钮 -> 选择 **“通过 URL 添加”**。
3. 输入订阅链接，并点击“添加日历”按钮。

### 📧 微软 Outlook (Outlook)
1. 登录 [Outlook 网页版](https://outlook.live.com/calendar/)。
2. 点击“添加日历” -> 选择“从 Web 订阅”。
3. 输入订阅链接和自定义日历名称，点击“导入”即可。

## 🌟 包含的内容

生成的日历每一天均为**全天事件**。事件标题格式为 `农历日期 · 节日/节气` (如：`农历五月初五 · 端午节`)。

点击事件，在“描述”中可以看到极其丰富的黄历和民俗信息：

*   **📅 农历基本日期**：如 `丙午(马)年 甲午月 丙辰日`
*   **☯️ 八字**：如 `丙午 甲午 丙辰`
*   **🔮 五行纳音**：如 `沙中土`
*   **🟢 宜**：当天适合进行的事情 (如：`解除、坏垣、治病` 等)
*   **🔴 忌**：当天应当避免的事情 (如：`移徙、入宅` 等)
*   **🎉 节日与节气**：包含传统农历节日、公历常用节日、传统二十四节气。
*   **🐎 冲煞**：冲煞生肖与煞方 (如：`冲(庚戌)狗，煞南`)
*   **🚪 每日胎神**：胎神占方 (如：`厨灶栖 外正东`)
*   **📌 彭祖百忌**：每日彭祖百忌 (如：`丙不修灶必见灾殃，辰不哭泣必主重丧`)
*   **🔱 建除十二神**：建除值日 (如：`开日`、`闭日` 等)
*   **🌌 每日九星**：完整的九星玄空、奇门与太乙运势 (如：`一白水 坎(正北) 天枢...`)
*   **⭐ 二十八宿**：宿名及吉凶，附带传统**宿歌诀**。
*   **✨ 吉神宜趋 & ⚠️ 凶神宜忌**：详细的神煞宜忌。

## ⚙️ 自动化机制

1. **自动更新**：依托 GitHub Actions，项目每天在北京时间零点（UTC 16:00）自动运行 Python 脚本生成最新的 `lunar_huangli.ics` 并提交回仓库。
2. **滚动范围**：始终保持前一年、今年、后一年的时间跨度（如 2026 年运行时，生成 2025、2026、2027 年的数据），文件大小适中，既能查询历史，又能预知未来，且加载极速。
3. **零外部 API 依赖**：使用 `lunar-python` 进行纯离线高精度天体历法计算，不消耗任何第三方 API 配额，100% 可靠，无失效风险。

## 💻 本地运行与调试

如果您想要在本地修改生成逻辑或手动生成日历：

1. 克隆本项目：
   ```bash
   git clone <your-repo-url>
   cd calendar
   ```
2. 创建虚拟环境并安装依赖：
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # macOS / Linux
   pip install -r requirements.txt
   ```
3. 运行生成脚本：
   ```bash
   python generate_calendar.py
   ```
4. 执行后会在目录下生成 `lunar_huangli.ics`，可直接双击导入系统日历中测试。
